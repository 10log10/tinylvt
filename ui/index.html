TinyLVT - Fair space allocation through land value taxation

TinyLVT helps communities allocate shared spaces fairly through simultaneous ascending auctions using land value taxation principles.

How it works

  • Simultaneous ascending auctions allocate spaces to highest-value uses
  • Proxy bidding lets you enter maximum values upfront
  • Multiple currency modes support different ownership models
  • Pay only the social cost of excluding others from a space
